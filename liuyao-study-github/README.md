# 六爻学习网站

第一版采用原生 HTML、CSS、JavaScript，无需安装依赖。

## 运行

在仓库根目录执行：

```bash
python3 -m http.server 4173 --directory "认知硬核库/六爻学习网站"
```

浏览器访问：`http://localhost:4173`

## 当前功能

- 八步完整学习路线
- 按 17 页讲义顺序组织的“讲义主课”
- 山医命相卜及四类占术定位
- 五行能量阶段与五行万物类象
- 带方向箭头的五行相生、相克交互图
- 河图先天数、洛书后天九宫及先后天八卦
- 八卦阴阳属性、先天序数、五行、方位、家庭与身体
- 十天干阴阳五行、五味五脏五常、二十四节气
- 春夏秋冬二十组五行细论
- 从五行关系自动推导六亲
- 旺相休囚死与十二支时空环
- 八卦爻画、方位、五行与类象互动
- 三钱起卦模拟、本卦与变卦识别
- 六亲分类与常见占问取用神
- 十二地支时空环
- 十二地支万物类象分类检索
- 按五行、类别和关键词筛选
- 地支翻卡训练
- 随机选择题
- 浏览与答题进度保存在本机

## 数据原则

- 当前以 [[认知硬核库/六爻/六爻预测法讲义20260606.pdf|老师讲义]] 为主。
- 每版讲义必须先完成逐页覆盖审计：[[认知硬核库/六爻学习系统/讲义覆盖审计-2026-06-06|2026-06-06 讲义覆盖审计]]。
- 后续古籍内容必须标记书名、章节、原文与现代解释。
- 老师讲义、古籍、整理推导、个人案例四类来源分开显示。
