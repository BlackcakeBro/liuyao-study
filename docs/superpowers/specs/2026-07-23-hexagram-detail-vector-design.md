# 八宫卦谱单卦卦象比例优化设计

## 目标

只调整新版网站八宫卦谱中“单卦详情”左栏的卦象展示：

- 卦象明显放大，但不改变卷轴、信息栏和卦名字号。
- 卦象以“单卦取象 · 某宫 · 某序位”和下方卦名为定位边界，在两行文字之间居中。
- 缩小无意义空白，保持上下间距紧凑、均衡。
- 六条爻线的宽高比例协调，爻线间距采用用户确认的 16px。
- 旧版网站以及八宫中的卦象卡片不受影响。

## 方案

### 1. 不再放大 Unicode 卦象字符

当前详情卦象直接使用 Unicode 六十四卦字符。字符的字体基线、内部留白和实际字形边界不可控，即使 CSS 盒子居中，视觉上仍可能偏上。

详情卦象改为六条 HTML 爻线直接绘制：

- 阳爻：一条完整横线。
- 阴爻：左右两段横线，中间留缺口。
- 每条爻厚度 10px。
- 相邻爻线间距 16px。
- 卦象总宽 176px。
- 阴爻中缝 24px。

八宫卦谱卡片中的小卦象继续使用现有 Unicode 字符，避免扩大本次变更范围。

### 2. 根据卦名生成六条爻线

从卦名解析上卦和下卦：

- 普通卦名取前两个自然象字，例如“泽雷随”解析为上卦“泽”、下卦“雷”。
- 八个纯卦取“为”字后的自然象并重复，例如“乾为天”解析为“天 / 天”。

八个自然象使用固定的从上到下爻序：

- 天：阳阳阳
- 泽：阴阳阳
- 火：阳阴阳
- 雷：阴阴阳
- 风：阳阳阴
- 水：阴阳阴
- 山：阳阴阴
- 地：阴阴阴

生成结果必须始终为六条爻线，并为图形添加包含卦名的无障碍标签。

### 3. 在两行文字之间建立紧凑展示区

卦象不再参照卡片上下边界，而是在上方详情标识和下方卦名之间建立独立展示区：

- 展示区高度 167px。
- 展示区上外间距 10px。
- 展示区下外间距 9px。
- 卦象在展示区内水平、垂直居中。
- 卦象本体高度约 140px，上下各保留约 13px 的视觉缓冲。

这样既保留呼吸感，也不会出现大片空白。

### 4. 动效

保留原有卦象呼吸阴影动效，但把动效施加到新的六爻图形容器上。动效只改变阴影，不改变尺寸、位置和爻线间距，避免布局抖动。

`prefers-reduced-motion: reduce` 环境继续关闭该动效。

### 5. 响应式

- 桌面端使用 176px 宽、10px 线厚、16px 间距。
- 中等屏幕和手机端适当缩小卦象宽度与线厚，但维持接近的宽高比例和明确的爻线间距。
- 卦名继续强制单行，不因卦象变化导致信息栏变高或变宽。

## 验证

### 自动测试

- 详情卦象不再输出 Unicode 字符，而是输出六条爻线。
- 乾、兑、离、震、巽、坎、艮、坤以及任一复卦均生成正确的六爻顺序。
- 64 卦全部生成且每卦恰好六条爻线。
- 桌面端样式包含 176px 宽度、10px 线厚和 16px 间距。
- 旧版网站与八宫卡片现有行为保持不变。

### 浏览器验证

- 新版直连进入八宫卦谱，点击宫位后再点击任一卦。
- 检查卦象位于上方详情标识和下方卦名之间，空白紧凑且上下协调。
- 切换不同阴阳组合的卦，确认六爻图形正确更新。
- 检查桌面端和手机端无重叠、裁切、横向溢出或布局跳动。
- 检查控制台无新增错误。
